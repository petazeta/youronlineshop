## About

This is a web app (Single Page Application) for e-commerce. Backend is written in PHP. It uses Mysql database system.


## Installation

If you prefer we offer FREE YOUR ONLINE SHOP installation and hosting service: ask at our email.

1. Upload the compressed file to your web server.
2. Unpack it to the destination folder (You may have to create the folder first).
3. Create a new mysql database.
4. Edit database settings at: backend/includes/config.php.
5. Run the app at the web browser and continue with the installation process.

## Getting started

If you have achieved to install your online shop you will see the shop main page at your browser. To make changes at the web you must log-in with some of the admin users. Regarding to the web content modifications there is the user webadmin. For modifications about the catalog content there is the user productsadmin. Other operations can be done by user systemadmin and ordersadmin. This last one can operate with the orders from customers.

Once editing some content press Intro or click outside of the editable area to save changes.

To reach the checkout process edition you should log in with systemadmin user and make an order as if you were a customer. Once you get to some checkout steps you should be able to edit some of the checkout options.

For more advanced system configuration you can edit file javascript/config.js.
